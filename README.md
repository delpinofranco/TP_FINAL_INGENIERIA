# TP_FINAL_INGENIERIA



## Descripción

El **Sistema de Gestión Interna de Talleres** es una aplicación diseñada para **optimizar y organizar** los procesos administrativos y operativos de un taller de herrería. Su objetivo principal es **centralizar toda la información** relacionada con clientes, presupuestos, materiales, trabajos en curso, pagos y fechas de entrega, permitiendo al taller llevar un **control eficiente** de sus operaciones diarias.

### Funcionalidades destacadas

-  **Registro y gestión de clientes**
-  **Control de presupuestos y trabajos realizados**
-  **Seguimiento de materiales y stock disponible**
-  **Registro de pagos y cálculo de ganancias**
-  **Gestión de sueldos del personal**

Con este sistema, el taller puede **reducir errores administrativos**, **mejorar la comunicación interna** y **aumentar la eficiencia** en la planificación y ejecución de los trabajos.
